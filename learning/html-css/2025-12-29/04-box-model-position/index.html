Box Model Demo

content-box (default)

width means content only. padding/border are added outside, so the rendered box becomes larger.

width: 220px

  • content: 220
  • padding: 20 × 2
  • border: 6 × 2
  • rendered = 220 + 40 + 12

border-box (practical standard)

width includes padding + border. Layout becomes predictable and matches design specs easily.

width: 220px

  • total: 220
  • padding/border are included
  • rendered = 220

Notes

Position 핵심 공식

  • absolute는 가장 가까운 position 조상(주로 relative)을 기준으로 한다.
  • 기준이 없으면 viewport(사실상 페이지) 기준으로 배치된다.
  • fixed는 항상 viewport에 고정된다.
  • sticky는 스크롤 임계점부터 fixed처럼 동작한다.

z-index & Stacking Context

  • z-index는 보통 position이 있어야 의도대로 동작한다.
  • stacking context가 다르면 z-index를 직접 비교할 수 없다.
  • 생성 예: opacity < 1, transform, filter, position + z-index
  • “z-index가 안 먹는다” = 대부분 stacking context 문제
